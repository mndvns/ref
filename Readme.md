# ref

Does the stuff I hate

## Usage

Via `ref --help`:

```sh
  
  Usage: ref [options] <command>

  Options:

    -h, --help                  output usage information
    -V, --version               output the version number
    -n, --name <string>         project name
    -d, --description <string>  project description
    -u, --username <name>       user legal name (e.g., Bob Ross)
    -e, --email <address>       user email address
    -s, --show-dependencies     render file showing module dependencies
    -p, --published <boolean>   whether or not the project is published
    -H, --hide-custom           hide custom commands
    --is-api                    
    --is-cli                    
    --is-stylus                 

  Commands:

    tag                          create and push annotated tag
    readme                       generate Readme.md
    license                      generate License
    render <template> <output>   render template to output

```
